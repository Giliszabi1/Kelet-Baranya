# Kelet-Baranya

Kelet-Baranya is a web and mobile application designed to bring together tourism and leisure information for the East Baranya region on a single platform.

The application helps tourists, local residents, and service providers easily discover events, attractions, restaurants, and accommodations.

the platform includes social features such as user profiles, friend connections, event participation, favorites, and personalized trip planning.

## Features

* User registration and login
* User profile and role/permission management
* Event management – create, delete, like, and join events
* Social features – add friends and view friends' event lists
* Interactive map features
* Personalized features – personal itinerary, custom hiking routes, and favorites
* Notification system
